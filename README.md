# 技能五子棋 2.0

基于 Web 的技能五子棋游戏，支持人机对战，内置多种特殊技能，AI 对手由豆包大模型驱动。

## 快速开始

### 前置要求

- Node.js >= 18

### 安装与运行

```bash
# 1. 安装依赖
npm install

# 2. 启动服务器
npm start
```

服务启动后访问 **http://localhost:3000** 即可进入游戏。

## 游戏功能

- **主菜单**：开始游戏、查看游戏说明
- **人机对战**：与 AI 对手进行五子棋对战
- **特殊技能**：在对局中可使用多种技能（如交换棋子、禁手等）
- **AI 对话**：游戏内集成 AI 对话功能

## 技术栈

- 前端：原生 HTML + CSS + JavaScript
- 后端：Express.js
- AI：豆包大模型（火山引擎）
